## Configuração do Ambiente Virtual e Rodar o Projeto

1. Execute o script para criar ou ativar a `venv`:
   ```bash
   ./create_and_activate_venv.sh
   ```

2. Com o ambiente virtual ativo, rode o projeto executando o seguinte comando:
    ```bash
    python training.py
    ```
